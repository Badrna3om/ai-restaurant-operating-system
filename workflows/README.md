# n8n Workflows

## Files

- `ai-restaurant-operating-system-public.json`  
  Main WhatsApp AI restaurant workflow.

- `close-order-session-public.json`  
  Resets the customer session after payment.

## Import

In n8n:

1. Open **Workflows**.
2. Select **Import from File**.
3. Choose the JSON file.
4. Reconfigure all credentials and identifiers.
5. Review every node before activating.

These files are sanitized and will not run until environment-specific configuration is added.
